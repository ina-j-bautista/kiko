# KIKO 🐋

### Bawat Aral, Bagong Tuklas

KIKO is an offline-first adaptive learning companion designed for Filipino learners in Grades 1–3. Guided by Kiko the Butanding, children progress through interactive lessons while the platform quietly adapts to their learning needs.

## Why KIKO?

Many educational platforms assume learners have a stable internet connection and can access content on demand. This assumption excludes many Filipino students who experience intermittent connectivity or rely on shared devices.

KIKO approaches the problem differently.

Instead of requiring AI to be available during every learning session, KIKO uses an **offline-first personalized curriculum loop**:

```
Learn Offline
      ↓
Store Progress Locally
      ↓
Sync When Connected
      ↓
Generate Personalized Lessons
      ↓
Download Content
      ↓
Learn Offline Again
```

The learner never waits for AI to respond. Personalization happens during connectivity windows, allowing the learning experience to remain fully functional even when internet access is unavailable.

## What Makes KIKO Different?

Most AI learning tools focus on real-time conversations.

KIKO focuses on **real-time learning and delayed personalization**.

Our system separates learning into three independent layers:

### Content Layer

Lessons are defined using a structured JSON schema that can be authored by teachers, downloaded from a server, or generated by AI.

### Progression Layer

Mastery tracking and curriculum progression run entirely on-device without requiring internet access or AI services.

### Personalization Layer

During synchronization, AI analyzes summarized learner performance and prepares the next batch of lessons tailored to the learner's current level and weak points.

This architecture ensures that personalization remains available even in low-connectivity environments.

## Current Prototype Scope

The proof-of-concept focuses on:

- Grades 1–3 learners
- DepEd-aligned foundational subjects
- Offline lesson delivery
- Local mastery tracking
- Personalized content synchronization
- Filipino and English content support
- Ocean-themed gamified experience featuring Kiko the Butanding

## Tech Direction

### Frontend

- Mobile-first experience
- Interactive lesson renderer
- Gamified progression system

### Content

- JSON-based lesson schema
- Modular exercise templates
- Downloadable lesson packages

### Personalization

- Local performance logging
- Weak-point detection
- AI-assisted lesson selection and generation during sync

## Vision

We believe personalized learning should not depend on continuous internet access.

KIKO delivers adaptive education through a model that brings AI into the background, allowing Filipino children to learn anytime, anywhere, while still receiving individualized support tailored to their needs.
